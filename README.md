 haa# EduLearn - Online Education System

## 📚 Project Overview

EduLearn is a simple Online Education System developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. The project provides a modern educational platform where students can log in, explore courses, and attempt an online sample examination.

This project was developed as part of a MERN Stack Internship Project.

---

## 🚀 Features

* 🔐 Student Login Page
* 🏠 Responsive Home Page
* 📖 Popular Courses Section
* 👨‍🏫 Why Choose Us Section
* ℹ️ About Platform
* 📩 Contact Form
* 📝 Online Sample Exam
* ⏱ Exam Timer
* ➡ Next & Previous Question Navigation
* 📊 Automatic Score Calculation
* 🎯 Exam Result Display
* 🚪 Logout Functionality
* 💾 MongoDB Database Integration (User Authentication)

## ⚙ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/EduLearn-Online-Education-System.git
```

### Open Project

```bash
cd EduLearn-Online-Education-System
```

### Install Dependencies

```bash
npm install
```

### Start MongoDB

Make sure MongoDB is running on:

```text
mongodb://127.0.0.1:27017
```

### Run the Server

```bash
node server.js
```

or

```bash
npm start
```
